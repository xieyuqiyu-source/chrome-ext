# Chrome Web Store Listing Draft

## Extension Name

我的笔记

## Short Description

轻量、专注、可随 Chrome 账号同步的侧边栏笔记扩展。

## Detailed Description

`我的笔记` 是一个围绕 Chrome Side Panel 设计的轻量笔记扩展。

它的目标很明确：打开快、记录快、查找快，并且不引入复杂账号体系。你可以直接在浏览器侧边栏中记录临时灵感、待办事项、网页摘录或工作备忘，并通过 Chrome 自带的同步能力在同一账号的多设备之间访问这些内容。

### 主要功能

- 侧边栏笔记工作流
- 笔记创建、查看、编辑、删除
- 基于 `chrome.storage.sync` 的同账号同步
- 关键词搜索与快速筛选
- 支持链接识别与点击访问
- 支持导出全部笔记文本
- 支持快捷键快速记录

### 适合谁

- 想要一个比传统网页笔记工具更轻量的记录方式
- 需要把零散灵感固定在浏览器侧边栏里
- 希望在同一个 Chrome 账号下跨设备同步轻量文本笔记

### 不做什么

- 不提供自建账号系统
- 不依赖独立云服务器
- 不适合保存大附件或超大规模资料库

## Privacy Disclosure Draft

- The extension stores user-created note content and note timestamps.
- Data is stored in `chrome.storage.sync`.
- The developer does not run a backend that receives note content.
- Data is not sold or used for advertising.
