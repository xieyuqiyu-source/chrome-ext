# 📝 我的笔记扩展

一个优雅的Chrome侧边栏笔记管理扩展，使用Manifest V3开发。

## ✨ 特性

- 🎨 **优雅的侧边栏界面** - 美观的渐变背景和现代化设计
- 📝 **智能输入区域** - 只有新增时才显示输入框，优化空间利用
- 🃏 **卡片式展示** - 漂亮的笔记卡片，支持悬停动效
- 🔍 **模态框查看** - 点击笔记卡片在模态框中查看完整内容
- ⚡ **快捷键支持** - Ctrl+N新建笔记，Ctrl+Enter保存，Esc取消
- 💾 **云端同步** - 使用Chrome存储API，数据跨设备同步
- 📤 **导出功能** - 一键导出所有笔记到剪贴板
- 🎯 **响应式设计** - 适配不同屏幕尺寸

## 🚀 安装方法

1. **打开Chrome扩展管理页面**
   ```
   chrome://extensions/
   ```

2. **启用开发者模式**
   - 点击右上角的"开发者模式"开关

3. **加载扩展**
   - 点击"加载已解压的扩展程序"
   - 选择包含此扩展的文件夹

4. **使用扩展**
   - 点击扩展图标打开侧边栏
   - 开始记录你的想法！

3. 开启"开发者模式"

4. 点击"加载已解压的扩展程序"，选择此文件夹
