优雅的侧边栏笔记管理扩展安装指南

📋 第一步：注意事项

这是一个侧边栏笔记扩展，需要Chrome 114+版本支持Side Panel API：

⚠️ 重要： 此扩展使用了Chrome Side Panel API，需要较新版本的Chrome浏览器。 如果你的Chrome版本较低，扩展可能无法正常工作。

🔧 第二步：打开Chrome扩展管理页面

在Chrome浏览器中：

  1. 点击右上角的三点菜单 ⋮
  2. 选择 更多工具 → 扩展程序
  3. 或直接在地址栏输入：
chrome://extensions/

🛠️ 第三步：启用开发者模式

在扩展管理页面：

  1. 找到页面右上角的 "开发者模式" 开关
  2. 点击开关启用开发者模式
  3. 页面会显示额外的开发者选项

📁 第四步：加载扩展

启用开发者模式后：

  1. 点击 "加载已解压的扩展程序" 按钮
  2. 浏览并选择包含扩展文件的文件夹
  3. 选择 ChromeExt 文件夹
  4. 点击 "选择文件夹"
🎉 成功！ 扩展现在应该出现在扩展列表中。点击扩展图标即可打开侧边栏笔记面板。

🔧 使用方法

基础操作：

  • 点击扩展图标 - 打开侧边栏笔记面板
  • 新建笔记 - 点击"✨ 新建笔记"按钮
  • 查看笔记 - 点击任意笔记卡片在模态框中查看
  • 编辑笔记 - 在查看模态框中点击"✏️ 编辑"按钮
  • 删除笔记 - 在查看模态框中点击"🗑️ 删除"按钮

快捷键：

  • Ctrl + N - 新建笔记
  • Ctrl + Enter - 保存笔记
  • Esc - 取消编辑或关闭模态框

高级功能：

  • URL识别 - 笔记中的网址会自动转换为可点击链接
  • 导出笔记 - 点击"📤 导出"按钮复制所有笔记
  • 清空笔记 - 点击"🗑️ 清空"按钮删除所有笔记

🛠️ 故障排除

常见问题：

  • 侧边栏不显示： 确保Chrome版本为114+，支持Side Panel API
  • 功能不工作： 检查浏览器控制台是否有错误信息
  • 数据丢失： 确保已登录Chrome账户以启用数据同步
  • 更新扩展： 修改代码后点击扩展管理页面的"重新加载"按钮
  • 权限问题： 确保扩展已正确安装并启用
